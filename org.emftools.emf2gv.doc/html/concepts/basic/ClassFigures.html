Class figures

Description

The ClassFigure contains the directives that will be used to render the diagram nodes.

Properties

The ClassFigure accepts the following properties :

CategoryNameDescriptionDefault literal
Appearance bodyBackgroundColorFigure's body background color#EEEEEE
headerBackgroundColorFigure's header background color#9DBDF9
labelEAttributeThe EClass attribute to use for the figure's label (the label that appears in the figure's header) 
labelStyleThe style of the class figure's label 
Container nestedFiguresEReferencesThe EReference to use to retrieve the EObjects to draw as nested figures 
Model eClassThe EClass that that this figure will represent on the graph
