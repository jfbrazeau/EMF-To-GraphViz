Attribute figures

Description

The AttributeFigure contains the directives that will be used to render the EObject's attribute values.

That figure looks like an UML attribute in a UML Class diagram, which means that it appears inside the EObject nodes, one above the other.

Properties

The AttributeFigure accepts the following properties :

CategoryNameDescriptionDefault literal
Appearance labelStyleThe style of the attribute figure's label 
Model eAttributeThe EAttribute that this figure will represent on the graph
