Depedencies

To use emf2gv in a standalone application, the following dependencies are required :

  • your EMF model plugin>
  • optional : <your EMF edit plugin> (if you want to automatically use the edit plugin's icons)
  • org.emftools.emf2gv.processor.core
  • org.emftools.emf2gv.graphdesc
  • org.emftools.emf2gv.util
  • org.emftools.validation.utils
  • org.eclipse.core.runtime
  • org.eclipse.emf.ecore
  • org.eclipse.emf.common
  • org.eclipse.emf.ecore.xmi
  • org.eclipse.equinox.common
  • org.eclipse.ocl.ecore
  • org.eclipse.ocl
  • org.eclipse.emf.edit
  • lpg.runtime.java

To get easily these libraries, it is possible to follow a method that is presented in this tutorial.
