Run configuration

Overview

Emf2gv provides a run configuration page that allows to change several generation parameters :

  1. Select in the Run menu > Run Configurations...
  2. Select the EMF To Graphviz section (on the left)
  3. It is then possible to create a new EMF To GraphiViz run configuration or to select an existing one


(Click to enlarge)

The emf2gv run configuration consists of two tabs :

  • the main tab
  • the filters tab

Main configuration tab

The main configuration tab allows to set :

  • the graphical description file (or you canask emf2gv to generate a default graphical description by checking the box)
  • the model to draw
  • the model element to select for the generation (root element)
  • the target image
  • if you want to keep the generated graphviz source file
  • if you want the model to be validated (which will make error and warning icons appear if required)

Emf2gv allows not to generate the diagram for the whole resource content. This is useful when your model contains a lot of contents. In order to restrict the EObjects to process, select the Select root element to process option and click Browse...
