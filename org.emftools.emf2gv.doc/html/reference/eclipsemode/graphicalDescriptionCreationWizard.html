Graphical Description Creation Wizard

To create a new graphical description resource, you need to use the creation wizard :

  1. Right click on your project, select New > Other...
  2. Select EMF Tools > Graphical Description Model, click Next

    (Click to enlarge)

  3. Select the filename (sample.graphdesc)

    (Click to enlarge)

  4. Select the model(s) you want to represent as a diagram ; we select the school model for the example

    (Click to enlarge)

  5. In the EClass selection page, select the EClass list you want to represent ; we select all the classes for the example

    (Click to enlarge)

  6. In the EReference selection page, select the EReferences list you want to represent. A special button allows to select the containment features ; we select all the EReferences for the example

    (Click to enlarge)

  7. Click Finish (there is no selection page for the EAttributes, all the EAttributes are selected by default)

    (Click to enlarge)

Once the graphical desription has been initialized, it is possible to start editing it with the Graphical Description Editor.
