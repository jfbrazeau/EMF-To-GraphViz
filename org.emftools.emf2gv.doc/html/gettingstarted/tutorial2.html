Tutorial #2 - Diagram Customization

Overview

In the first tutorial, we've seen how to let emf2gv generate a default diagram for a given model. In this tutorial, we will see how to start customizing a diagram.

Emf2gv takes as input the following data :

  • A model file
  • A graphical description corresponding to the EPackages used in the model file

If we let emf2gv generate a default diagram as in the first tutorial, a default graphical description is generated. If we want to start to customize the diagram, we have to create a graphical description and modify it.

This tutorial is based on a School / Classroom / Student model provided by the EMF Tools Samples feature which is available on the update site of the EMF Tools Project (see the installation page for more details).

Graphical Description Creation Steps

  • Create an empty project in your eclipse workspace named MyProject (or use the on you created in the first tutorial)
  • Create a graphical desccription named sample.graphdesc by using the creation wizard (follow the instructions of this page

  • Right click on the sample.graphdesc file, select Run as > Emf To Graphviz Transformation

Diagram Customization Steps

Diagram Orientation

  • Edit the graphical description (sample.graphdesc)
  • Select the GVFigureDescription node
  • Set the Orientation property to TopDown
  • Right click on the sample.graphdesc file, select Run as > Emf To Graphviz Transformation

You should get the following result :


(Click to enlarge)

Nodes Alignment

  • Edit the graphical description (sample.graphdesc)
  • Select the GVFigureDescription node
  • Set the Align Same EClasses property to true
  • Right click on the sample.graphdesc file, select Run as > Emf To Graphviz Transformation
